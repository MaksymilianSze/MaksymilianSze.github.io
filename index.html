Bootcamp 2020: Mandelbrot Set

My first programming project, concluding the 2 week Programming Bootcamp at City (2020). This processing project uses the the understanding of iteration and selection, as well as, some maths compenents such as imaginary numbers, complex numbers, diverging and converging numbers. By iterating the equation f(z)=z^2+c and colouring a pixel based on whether the result tends to infinity I created a nice looking fractal. I also created a zoom feature so you can look at the patterns that get smaller and smaller in an interactive way using the mouse. I used floating point numbers to store my values for my project and one limitation of this is that when you zoom in large amounts into the fractal you start to see the inaccuracies (it gets blurry and pixelated)as the number get smaller and smaller.

Check out the code on Github: https://github.com/MaksymilianSze/adbt182
